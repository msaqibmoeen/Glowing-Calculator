# Glowing-Calculator
This project is a simple web-based calculator built using HTML, CSS, and JavaScript. It provides basic arithmetic functionality with a clean and visually appealing glowing user interface.

🚀 Features
Perform basic operations:
Addition (+)
Subtraction (−)
Multiplication (×)
Division (÷)
Interactive button-based input
Clear display functionality
Responsive layout for different screen sizes
Stylish glowing UI using CSS

🧱 Project Structure
HTML (index.html)
Defines the structure of the calculator
Includes a display screen and buttons for numbers and operations
CSS (style.css)
Handles layout and design
Adds glow effects and grid alignment for buttons
JavaScript (script.js)
Controls calculator functionality:
Appends values to the display
Clears the display
Calculates results using user input

🖥️ How It Works
Users interact with the calculator by clicking buttons.
Each button triggers a JavaScript function:
Numbers and operators are added to the display.
The C button clears the screen.
The = button evaluates the expression and shows the result.
⚠️ Note
The calculator uses JavaScript’s built-in evaluation method (eval) to compute results. While it works for basic operations, it should be used carefully in production environments.

🛠️ Technologies Used
HTML5
CSS3 (Grid & Effects)
JavaScript (DOM Manipulation)
